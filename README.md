# Schwenck

> *"The only thing more absurd than training an LLM exclusively on the works of Gilbert & Sullivan is doing it in R!"*

My goal in this project is to build a tiny, character-level GPT, trained on Gilbert & Sullivan libretti, using R and [`torch`](https://torch.mlverse.org/). If all goes well, the result will be deliciously absurd on multiple levels.
